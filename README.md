## Описание проекта

Random Task Generator - это desktop-приложение на Python с графическим интерфейсом для генерации случайных задач по трём категориям: учёба, спорт и работа. Приложение помогает разнообразить повседневные дела и добавляет элемент случайности в планирование.

### Основные возможности

- Генерация случайных задач из предопределённого списка
- Добавление собственных задач с выбором типа
- Сохранение истории сгенерированных задач в JSON
- Фильтрация истории по типу задачи
- Просмотр статистики использования
- Валидация ввода (проверка на пустые строки)
- Автоматическая загрузка истории при запуске
- Красивый и интуитивно понятный интерфейс

## Требования

- Python 3.7 или выше
- Стандартные библиотеки Python (tkinter, random, json, datetime, os)
